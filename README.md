# 📖 About The Project

**Video Rewind Studio** is a browser-based web application that enables users to reverse both **video** and **audio** without requiring any software installation or server-side processing.

The application is powered by **FFmpeg.wasm**, allowing all video processing to happen directly inside the browser. Since files never leave the user's device, the tool provides complete privacy while delivering fast and efficient performance.

This project was developed as a **Project-Based Learning (PBL)** project to demonstrate modern client-side multimedia processing using web technologies.

# ⚙️ How It Works

1. Upload a supported video.
2. Preview the original video.
3. Click **Reverse Video**.
4. FFmpeg.wasm processes the file locally.
5. Both the video frames and audio track are reversed.
6. Preview the processed video.
7. Download the final output.
